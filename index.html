Servicios

¿Qué hago por los clientes?

Establecer una comunicación estrecha con el cliente en cada una de las fases del desarrollo de la página web a la medida que son la planeación, diseño y programación de la página web cubriendo todos los objetivos esperados por los clientes, así como soporte técnico ante situaciones de software y hardware

Planeación

Se le solicita al cliente un cuestionario para la elaboración de su página web una vez teniendo toda la información del cuestionario se traza un plano de como se ubica el logo, menú, imágenes y la información solicitada.

Diseño

Una vez que el cliente acepta el plano se elabora un dibujo real bien definido de la página web para verificar cambios finales y se definen las funciones que desea el cliente en la página web.

Programación

Aceptados el plano y el diseño por el cliente se desarrolla la programación de la página web final, con sus respectivas funcionalidades que el cliente defina.

Soporte Técnico

Se brindan alternativas de solución con asesoría y atención personalizada para los casos de software y hardware.
